La web app fornisce un elenco dei Luoghi per Eventi (sale per conferenze, aule per formazione, teatri, ecc) presenti nel territorio dell’Altovicentino.

Tramite la webapp è possibile

  • cercare un Luogo per Eventi tramite una delle seguenti modalità
    • per parola chiave, inserendo una parola (ad esempio “teatro”) e premendo il pulsante "Ricerca"
    • per parola chiave e per vicinanza alla propria posizione, inserendo una parola e premendo il pulsante "Geo Ricerca"
    • per categoria, scegliendo uno dei pulsanti categoria disponibili
  • visualizzare le informazioni relative ad un dato Luogo per Evento.
